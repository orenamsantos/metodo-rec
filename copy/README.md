## Regras de copy

- **Espanhol neutro LATAM** — evitar regionalismos fortes (ej: "vos", "tío", "chévere" só quando intencional)
- **Tom editorial, feminino, terapêutico** — não agressivo, não "compra agora", não emoji-spam
- **Pronominal próximo** — "tú", não "usted" (intimidade, mas com respeito)
- **Italicos em palavras-chave** — emocionais, marcadas no JSX com `<Em>` (cor dourada)
- **Números específicos** — `73,4%`, `12.847 mujeres`, `78%` — credibilidade
- **Verbos no presente** — urgência, "estás", "sientes", "necesitas"
